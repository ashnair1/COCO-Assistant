    get_imgid_dict(ann) ¶

    Returns a dictionary with img ids as keys and img filenames as values

    Source code in coco_assistant/coco_visualiser.py 
    def get_imgid_dict(ann):
    """
    Returns a dictionary with img ids as keys and img filenames as values
    """
    return {item[1]["file_name"]: item[0] for item in ann.imgs.items()}
